# PWA Icons

This directory should contain the following PWA icons:

- `pwa-192x192.png` - 192x192 PNG icon for the manifest
- `pwa-512x512.png` - 512x512 PNG icon for the manifest
- `apple-touch-icon.png` - 180x180 PNG icon for iOS home screen
- `favicon.ico` - Standard favicon

You can generate these icons from a source image using online tools like:
- https://favicon.io/favicon-generator/
- https://www.pwabuilder.com/
- Or use tools like RealFaviconGenerator

For development, you can use placeholder images, but for production, proper icons are recommended.
